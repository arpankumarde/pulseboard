# Pulseboard

Pulseboard is a micro-SaaS platform designed to help solo builders and developers stay consistent with their work habits. It provides tools to log focused work sessions, track progress over time, and gain insights into productivity patterns. With a clean and distraction-free interface, Pulseboard empowers users to build better habits and optimize their workflows.

## Features

- **Session Logging:** Quickly log work sessions with title, category, and duration.
- **Dashboard:** View recent activity, track streaks, and monitor daily focus time.
- **Analytics (Pro):** Unlock weekly insights, category breakdowns, and streak analysis.
- **Subscription Plans:** Free plan with basic features; Pro plan for advanced analytics.
- **Focus Streaks:** Maintain momentum by tracking daily activity streaks.
- **Simple Onboarding:** Get started in minutes with a user-friendly interface.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS
- **Backend:** FastAPI, SQLModel
- **Database:** PostgreSQL
- **Payments:** Razorpay for subscription management
- **State Management:** React Hooks, Context API
- **Authentication:** JWT-based authentication
